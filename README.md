## ğŸ”§ Key Walkers

### `createGraph`
Initializes the graph database with cities, routes, and bus schedules.

### `filterBuses`
Searches for buses based on:
- Source and destination cities
- Time range (start/end time)
- Bus type (normal, semi-luxury, luxury)
- Route type (direct, via)

### `findRoutes`
Discovers all available route numbers between two cities.

### `agentAI`
Main AI orchestrator that:
1. Classifies user intent (general chat vs bus search)
2. Routes to appropriate specialist walker
3. Generates natural language responses

### `computeBusTimeAgent`
Uses **ReAct methodology** with tools:
- `compute_average_speed()` - Returns speed by bus type
- `compute_stop_delay()` - Calculates delay from intermediate stops
- `compute_number_of_intermediate_stops()` - Finds stops on shortest route
